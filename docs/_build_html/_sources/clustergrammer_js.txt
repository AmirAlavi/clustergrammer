Clustergrammer.js
-----------------
The front-end library Clustergrammer.js builds the interactive heatmap visualization in SVG using the visualization library D3.js. Clustergrammer is built using Webpack Module Developer.

Dependencies
============

- D3.js
- JQuery
- Underscore
