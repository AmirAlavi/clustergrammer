Input Matrix Format
-------------------
Clustergrammer.py takes as input:

#. a tab-separated matrix file with unique row and column names as input
#. a Pandas DataFrame

The simplest file format is shown here:
::

       Col-A   Col-B   Col-C
Row-A   0.0    -0.1     1.0
Row-B   3.0     0.0     8.0
Row-C   0.2     0.1     2.5
